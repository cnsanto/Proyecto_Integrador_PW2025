# FutMatch:

FutMatch es una aplicación web orientada a la organización y gestión de partidos de fútbol amateur. El sistema permite crear partidos, administrar reservas, asignar jugadores a equipos y controlar el estado de cada encuentro, centralizando toda la información en una base de datos relacional bien estructurada. El proyecto fue desarrollado con foco en buenas prácticas, normalización de base de datos, lógica de negocio clara y una interfaz web funcional, pensada tanto para usuarios finales como para facilitar el mantenimiento y la escalabilidad del sistema.

Funcionalidades básicas relevantes:

* Creación y gestión de partidos con fecha, horario, ubicación y estado.
* Sistema de reservas asociado a partidos.
* Asociación de usuarios/jugadores a reservas y partidos.
* Asignación de jugadores a Equipo A o Equipo B.
* Control de cupos por equipo y cantidad de participantes.
* Validaciones de negocio (por ejemplo, partidos abiertos/cerrados o finalizados).
* Consulta de partidos disponibles y detalles de cada encuentro.
* Uso de relaciones entre tablas mediante claves foráneas.
* Base de datos normalizada para evitar duplicación de datos.
* Interfaz web construida con HTML, Bootstrap y JavaScript.
* Lógica del lado del servidor implementada en PHP.
* Manejo de errores y validaciones tanto en frontend como backend.
